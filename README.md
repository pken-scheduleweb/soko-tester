# [P研] 倉庫スケジュール — 使い方ガイド

## サイトページ

https://pken-scheduleweb.github.io/soko/

---

## カレンダーの見方

サイトを開くと、**今週（火曜〜月曜）** の倉庫利用スケジュールが表示されます。

- 予定ブロックをタップ／クリックすると、詳細（名前・時間）が確認できます
- 同じ名前の予定には同じ色が自動で付きます
- スマホではカレンダーを左右にスワイプして全曜日を確認できます
- 月曜日の18:00以降から次週の予約ができるようになっています

---

## 予定を追加する

1. **「+ 予定を追加」** ボタンを押す
2. 以下の項目を入力する

   | 項目 | 説明 |
   |------|------|
   | 名前 | 自分の名前 |
   | 曜日 | 利用する曜日 |
   | 開始・終了時刻 | 10〜20時の範囲で10分刻み |
   | PIN | 4桁の数字（削除・編集時に使用）|

3. **「追加する」** を押して完了

> 「ブロックの色」から好きな色を選べます。未選択の場合は名前から自動で色が決まります。

> 特定のブロックを左クリック、タップを行うと自動的に日時の項目が入力された状態で表示されます

> **注意：** 他の予定と時間が重なっている場合は警告が表示されます。重複していても問題なければそのまま追加できます。

---

## 予定を削除する

1. 削除したい予定ブロックを **右クリック**（スマホは **タップ**）
2. メニューから **「削除」** を選ぶ
3. 追加時に設定した **PIN（4桁）** を入力する
4. **「削除する」** を押して完了

> **PIN を忘れた場合：** 管理者に連絡してください。管理者はPINなしで削除できます。

---

## 画像保存

「**画像保存**」ボタンを押すと、今週のカレンダーを画像として保存できます。

- **スマホ：** 写真アプリへの共有シートが開きます
- **PC：** 自動でダウンロードされます

## 通知機能（任意）

アカウントを作成してログインすると、予定の時刻や他の人の予約追加をメールで受け取れます。

### アカウントを作る

1. **「新規登録」** ボタンを押す
2. メールアドレスとパスワード（4文字以上）を入力する
3. **「登録する」** を押す

### ログイン

1. **「ログイン」** ボタンを押す
2. 登録したメールアドレスとパスワードを入力する
3. **「ログイン」** を押す

### 通知の設定

ログイン後、**「通知設定」** ボタンを押すとON/OFFを切り替えられます。

| 通知の種類 | 内容 |
|-----------|------|
| 自分の予定の時刻通知 | 予定の **1時間前** と **開始時刻** にメールが届く |
| 予定追加の通知 | 誰かが新しく予定を追加したときにメールが届く |

設定を変えたら **「保存」** を押してください。

### ログアウト

ヘッダーの **「退出」** ボタンを押すとログアウトできます。

### アカウントの削除

**「登録解除」** ボタンを押し、確認画面でメールアドレスを入力すると削除できます。

## ボタンが隠れているとき（スマホ）

画面上部のボタン行は **横にスワイプ** すると隠れているボタンが見えます。

## よくある質問

**Q. PINを忘れてしまいました**
A. 管理者に連絡してください。管理者はPINなしで削除できます。

**Q. 通知メールが届きません**
A. 以下を確認してください。
- ログインしているか
- 「通知設定」で通知が ON になっているか
- 迷惑メールフォルダに入っていないか

**Q. 過去の予定が見えません**
A. 一般ユーザーは今週のみ表示されます。過去の予定を確認したい場合は管理者に相談してください。

**Q. 予定ブロックの色を変えたい**
A. 予定追加フォームの「ブロックの色」から好きな色を選べます。「自動に戻す」で名前ベースの自動色に戻せます。
